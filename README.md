## About Project

### Target
+ 쓰레기통이 꽉 채워져 있거나 버릴 곳이 없어 불편함을 느끼는 사용자
+ 플로깅 활동을 하는 사용자
+ 환경에 관심있는 사용자

<br/>

### Skills
+ JAVA
+ Android Studio
+ DB: Firebase
+ AI: Teachable Machine
